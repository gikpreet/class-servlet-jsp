= CGI (Common Gateway Inteface)

CGI는 동작 웹의 시작점이라 할 수 있는 기술로, 웹 서버가 외부 프로그램을 실행할 수 있도록 해주는 인터페이스 명세이며, C, C++, java, go 등 다양한 언어에서 웹 애플리케이션을 작성할 수 있도록 하는 기술입니다. 이 단원에서는 CGI에 대해 알아봅니다.
