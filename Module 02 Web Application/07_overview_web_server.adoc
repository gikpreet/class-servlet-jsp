= Web Server 개요

* HTTP 또는 HTTPS를 통해 웹 브라우저 또는 프로그램에서 요청에 응답하여 요청 결과를 전송하는 서비스 프로그램
** HTTP/HTTPS 프로토콜을 사용
** 웹 브라우저 요청에 대한 응답을 전송
* 응답 유형
** HTML document
** 이미지, 파일
** HTTP 상태 코드
** 요청 결과(JSON, XML 등)
